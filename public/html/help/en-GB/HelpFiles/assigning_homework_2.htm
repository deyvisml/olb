Assign homework

These features are currently only available when using oxfordlearnersbookshelf.com website.

You can now assign homework to your students by copying a link to a book or copying a link to a page using the copy link buttons.

Note: To assign homework, teachers need to have the same book as the student. Students need a licence for the book to see their homework assignments.
 
You'll need to copy a link to a page in a student's book to assign homework.

Watch our YouTube video on using links to assign homework

Copying a link to a book

You can copy a link to a book from your bookshelf using the 'Copy link to this book' button.

 

Copying a link to a page

You can copy a link to a page from your books using the 'copy link' icon found on each page.
