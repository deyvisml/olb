System requirements for the desktop app
Operating System

•Windows 10 or later (see Note)

•Mac OS 10.15 or later

•Linux Ubuntu 16.04 LTS (64-bit) or later

Hardware

•CPU: 2 GHz and above

•Physical Memory: 1024 MB or more available

•Disc Space (see Note)

oWindows: 151 MB of disc space to install the app

oMac: 167 MB of disc space to install the app

oLinux: 300 MB of disc space to install the app

•Minimum screen resolution: 1024 x 768

 

Note
 
Windows computers need Microsoft .NET Framework 3.5 installed and enabled to download books. If you do not have .NET Framework 3.5, you will be asked to install it and restart the app before you can download books from the Cloud.
 
You'll also need additional space for each e-book download. Each e-book download is between 30MB and 1GB, depending on the title.

Using an iPad or Android device

You can use the Oxford Learner's Bookshelf app for iPad or Android devices. Find them in the App Store and on Google Play.
