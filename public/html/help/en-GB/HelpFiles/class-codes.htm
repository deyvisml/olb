Create class codes

When you create a class on Oxford English Hub you get a code for that class.

Students can use this code to join your class using the Join class button on their Bookshelf.

Sign in to Oxford English Hub with your Oxford ID and register your organization to get started.

After you register your organization you can:

•create classes

•assign course materials to students

•track student progress

Learn more
