Add books with an access code

From your Bookshelf, you can see the books you have access to or add new books using an access code.

Note:
You'll need to be connected to the internet to add or download a book.

To enter an access code:

1.Sign in using your Oxford ID username and password.

2.Click Add books:

3.Enter your 12-digit access code and click the Add button.

Your books will be added to the Bookshelf. Once you have added a book, you'll need to download it to your computer or laptop to use it.
